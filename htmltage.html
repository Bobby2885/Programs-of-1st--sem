Basic HTML Structure Tags

< html > The root element of an HTML document.
< head > Contains meta-information about the document (e.g., title, links to CSS, etc.).
< title > Sets the title of the document shown in the browser tab.
< body > Contains the content of the HTML document (e.g., text, images, links).

Text Formatting Tags

< h1 >to < h6 > Heading tags, with < h1 >being the highest level (largest) and < h6 >the lowest (smallest).
< p > Paragraph tag used for text blocks.
< br > Line break; moves content to the next line.
< hr > Horizontal rule; creates a thematic break or a horizontal line.

Inline Formatting Tags

< strong > Makes text bold.
< em > Emphasizes text (italic).
< b > Bold text without implying importance.
< i > Italic text without implying emphasis.
< mark > Highlights text.
< small > Displays text in a smaller font.
< sub >and < sup > Subscript and superscript text.

Links and Media Tags

< a > Anchor tag used for hyperlinks.
<img >Image tag for displaying images. Attributes include src (source) and alt (alternative text).
<audio> and <video> Tags for embedding audio and video.

List Tags

<ul> Unordered list (bullet points).
<ol> Ordered list (numbered).
<li> List item used inside <ul> or <ol>.

Table Tags

<table> Creates a table.
<tr> Table row.
<th> Table header cell.
<td> Table data cell.

Form Tags

<form> Container for form elements.
<input> Input field.
<label> Label for input elements.
<textarea> Multi-line text input.
<button> Clickable button.
<select> and <option> Dropdown list.

Semantic Tags

<header> Defines a header section.
<footer> Defines a footer section.
<section> Defines a section in a document.
<article> Self-contained piece of content.
<aside> Sidebar content related to the main content.
<nav> Navigation links.
<main> Main content of the document
