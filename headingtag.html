HTML Headings HTML headings are defined with the <h1 > to <h6 > tags.

This is an H1 Heading

The <h1> tag is used for the main heading and should be used only once per page.

This is an H2 Heading

The <h2> tag is used for subheadings and can be used multiple times.

This is an H3 Heading

The <h3> tag is used for sub-subheadings under <h2> tags.

This is an H4 Heading

The <h4> tag is often used for sections within <h3> sections.

This is an H5 Heading

The <h5> tag is used for sub-sections within <h4> sections.

This is an H6 Heading

The <h6> tag is used for the lowest level of headings, often used for minor details.
